HTML

¿QUE ES?

HTML es el lenguaje con el que se define el contenido de las páginas web. El lenguaje HTML sirve para describir la estructura básica de una página y organizar la forma en que se mostrará su contenido, además de que HTML permite incluir enlaces hacia otras páginas o documentos. Hay que mencionar que el HTML no es un lenguaje de programación, ya que no cuenta con funciones aritméticas, variables o estructuras de control propias de estos lenguajes, por lo que el HTML únicamente sirve para crear páginas web estáticas.

¿QUE SON LAS ETIQUETAS HTML?

El lenguaje HTML está compuesto por etiquetas. Las etiquetas se utilizan para describir algo que se quiere representar en una página web, su apariencia

TIPOS DE ETIQUETAS

El HTML tiene definidas gran variedad de etiquetas para distintos usos en este artículo te vamos a explicar las más esenciales. Etiquetas de apertura. Etiquetas de cierre. Son lo mismo que las etiquetas de apertura, pero estas indican donde finaliza el elemento.
Etiquetas de título. Etiquetas de cuerpo. Esta etiqueta indica la parte del cuerpo o desarrollo del texto. Etiqueta de sección inferior.
Etiqueta de sección superior. Esta etiqueta indica la parte superior del texto o encabezado. Etiqueta de título. Etiqueta de subtítulo. Etiqueta de apartado. Etiquetas de párrafo. Etiquetas de negrita. Etiquetas de cursiva. Etiquetas de imagen. Esta etiqueta no necesita una de cierre. Etiquetas de tachado. Etiquetas de enlace. Cada una de las etiquetas debe acabar con su equivalente de cierre. Debemos tener claro que es muy importante que por cada etiqueta que abramos, deberemos incluir la correspondiente etiqueta de cierre. Así conseguiremos un código HTML bien formado y que los navegadores puedan interpretar sin espacio a error ni ambigüedad.
